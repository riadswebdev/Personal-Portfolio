## 📌 Project Overview
**Project Type:** Personal Portfolio / Landing Page

This is a professional, beautifully crafted personal portfolio built to showcase frontend development skills, projects, and professional experience. It leverages the power of Next.js for performance, Tailwind CSS for a sleek glassmorphism design, and GSAP & Lenis for an immersive, buttery-smooth animated scrolling experience.

---

## 🛠️ Technologies Used

*   **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
*   **Library:** [React 19](https://react.dev/)
*   **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
*   **Animations:** [GSAP](https://gsap.com/) & `@gsap/react`
*   **Smooth Scrolling:** [Lenis](https://lenis.studiofreight.com/)

---

## ✨ Key Features

*   ⚡ **Lightning Fast:** Optimized with Next.js for superior performance and SEO.
*   🎬 **Immersive Animations:** Scroll-triggered GSAP animations on text, cards, and sections.
*   🌊 **Smooth Scrolling:** Integrated Lenis for a premium, native-feeling scroll experience.
*   💎 **Modern UI/UX:** Clean, responsive glassmorphism design system.
*   📱 **Fully Responsive:** Adapts flawlessly across mobile, tablet, and desktop screens.
*   📬 **Working Contact Form:** Integrated with Formspree for seamless direct messaging.

---

## 🧩 Sections / Components

*   **Top Navigation Bar:** Includes smooth anchor links and a nested dropdown for Skills.
*   **Hero Section:** Engaging introduction with animated typography.
*   **About Me:** Background details, statistics, and career overview.
*   **Skills & Qualifications:** Detailed breakdown of technical stacks and education.
*   **Projects Showcase:** Displaying recent works and case studies.
*   **Contact Section:** Direct contact links (Email, Phone, WhatsApp) and an active form.
*   **Footer:** Brand logo and quick links.
